Gotit 开放接口文档

  • 更新时间: 2014-04-15

  • 数据格式均为JSON

  • 接口说明:

    1. 由于中文验证码的存在, 接口或许有点另类, 希望见谅.
    2. 验证类接口需要登录后才能使用, 如果仅查询单项信息推荐使用下面的成绩类或课表类接口
    3. 希望开发者在使用接口前先通过邮件联系作者, 如果有很好的建议或者哪些地方不理解也可以通过邮件联系.

验证类

登录

  • 功能
    登录后可以直接获取下面所有接口的结果, 仅需要提供用户识别码(UID)
  • 接口地址
    /user/login.json

  • 请求方式
    GET + POST

    • 使用说明

      1. GET: 先进行一次GET获取 UID 值(用户识别码), 用户的唯一标识.
      2. 通过UID获取验证码, 验证码链接为http://api.gotit.asia/zheng/checkcode.gif?uid=[UID].
      3. POST: POST的数据有: 学号, 密码, 中文验证码(utf-8), 用户识别码UID

    • GET 结果示例

        {
        "status": {
            "code": 200,
            "message": "Success",
        },
        "data": {
            "uid": "user_067e11ac790844c08e068c96fb5b023b"
        }
    }
    • POST 示例

    参数: data

        {
        "uid": "user_067e11ac790844c08e068c96fb5b023b",
        "xh": "1111123456",
        "pw": "password",
        "verify": "疫讨",
    }

  • 返回值示例

    {
        {
            "status": {
                "code": 200,
                "message": "Login Success",
            },
            "data": {
                "uid": "user_067e11ac790844c08e068c96fb5b023b"
            }
        }
    }
